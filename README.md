# Core

An awesome app built on the [Denali framework](http://denalijs.org/).


## Getting Started

1. Clone the repo down
2. `npm install -g denali denali-cli`
3. `npm install`
4. `rethinkdb`
5. In another terminal `denali server`
6. Hit [localhost:3000](http://localhost:3000)


## Tests

```sh
$ denali test
```
